**Read & Set ACLs**

\# setfacl -m user:\<username\>:\<permissions\> \<file\>

\# setfacl -m group:\<groupname\>:\<permissions\> \<file\>

\# setfacl -d -m user:\<username\>:\<permissions\> \<file\>

\# getfacl \<file\>

\# getfacl -t \<file\>
